# Ashland: Last Escape — 游戏设计规格文档

## 1. 概述

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 游戏名 | Ashland: Last Escape（灰烬大陆：最后的渡口）|
| 类型 | 2D 俯视角像素风末日生存 |
| 目标时长 | 2 小时（完整通关）|
| 目标平台 | itch.io / Steam |
| 引擎 | Godot 4.x |
| 开发方式 | 一个人 + AI 工具链 |

### 1.2 一句话描述

> 末日降临，你是最后一批幸存者，必须穿过三片废土地带，找到通往救赎彼岸的最后渡口。

### 1.3 目标玩家

- 喜欢像素风独立游戏的玩家
- 末日/生存题材爱好者
- 喜欢探索和资源管理的玩家
- 轻度硬核（不需要魂类操作）

## 2. 世界观

### 2.1 背景故事

公元 21XX 年，一场未知的灾难降临地球。大气被尘埃覆盖，辐射蔓延，文明崩塌。主角是一名普通幸存者，在废墟中醒来，必须在资源彻底枯竭前，穿过三片废土地带，找到传说中"最后的渡口"——那是人类最后的撤离点。

### 2.2 三片区域

#### Ch1：沦陷之城（Ash City）
- **环境**：被遗弃的城市废墟，高楼残骸，废弃车辆
- **色调**：灰蓝 + 暗红
- **特征**：教学关，相对安全，有基础资源
- **主要威胁**：零散丧尸，低级掠夺者

#### Ch2：辐射荒原（Blast Wastes）
- **环境**：开阔的荒漠，被核弹犁过的土地，变异生物出没
- **色调**：土黄 + 辐射绿
- **特征**：资源匮乏，辐射伤害，昼夜温差
- **主要威胁**：变异兽群，沙尘暴

#### Ch3：避难者聚落（Refuge Settlement）
- **环境**：幸存者建立的地下聚落，有商店、NPC
- **色调**：暖黄 + 钢铁灰
- **特征**：相对安全区，可交易、可接支线
- **主要威胁**：少量敌人，NPC 任务风险

#### Ch4：最后的渡口（Last Harbor）
- **环境**：海边的废弃港口，有巨型 Boss 镇守
- **色调**：深蓝 + 金属灰
- **特征**：最终关，Boss 战
- **主要威胁**：末日守卫（Boss）

## 3. 核心系统

### 3.1 移动系统

| 属性 | 值 |
|------|------|
| 移动方式 | WASD，8 向移动 |
| 基础移速 | 120 px/s |
| 碰撞 | TileMap 碰撞体 |

### 3.2 资源系统

#### 资源种类

| 资源 | 图标颜色 | 用途 |
|------|---------|------|
| 木材 (Wood) | 棕色 | 合成基础材料 |
| 石材 (Stone) | 灰色 | 合成武器/工具 |
| 食物 (Food) | 红色 | 恢复饥饿值 |
| 医疗包 (Medkit) | 绿色 | 恢复生命值 |
| 零件 (Parts) | 蓝色 | 合成高级装备 |
| 弹药 (Ammo) | 黄色 | 远程武器消耗 |

#### 采集方式

- 靠近资源物 → 按 E 采集
- 采集时间：0.5 秒
- 资源点有数量限制（采集 N 次后消失）

### 3.3 生存属性

| 属性 | 最大值 | 消耗速度 | 死亡条件 |
|------|--------|---------|---------|
| 生命值 (HP) | 100 | 战斗/辐射 | HP ≤ 0 → 死亡 |
| 饥饿值 (Hunger) | 100 | 每 10 秒 -1 | Hunger ≤ 0 → 持续掉 HP |
| 体力 (Stamina) | 100 | 冲刺时消耗 | Stamina = 0 → 无法冲刺 |

### 3.4 合成系统

#### 合成配方

| 物品 | 材料 | 效果 |
|------|------|------|
| 简易绷带 | 布料 x2 | 恢复 HP 20 |
| 烤肉 | 食物 x1 + 木材 x1 | 恢复 Hunger 30 |
| 木棍 | 木材 x2 | 近战武器，伤害 10 |
| 石斧 | 木材 x1 + 石材 x2 | 采集效率 +50% |
| 急救包 | 医疗包 x2 + 零件 x1 | 恢复 HP 50 |
| 弓箭 | 木材 x3 + 零件 x2 | 远程武器，伤害 15 |
| 铁剑 | 石材 x3 + 零件 x3 | 近战武器，伤害 25 |
| 防辐射药 | 零件 x2 + 食物 x1 | 辐射抗性 +30 秒 |

### 3.5 战斗系统

#### 武器

| 武器 | 伤害 | 攻速 | 范围 |
|------|------|------|------|
| 拳头 | 5 | 快 | 近身 |
| 木棍 | 10 | 中 | 近身 |
| 石斧 | 15 | 中 | 近身 |
| 铁剑 | 25 | 快 | 近身 |
| 弓箭 | 15 | 慢 | 远程 |
| 手枪 | 20 | 快 | 远程 |

#### 敌人 AI

- **巡逻状态**：在固定路径上移动
- **警戒状态**：发现玩家后追击
- **攻击状态**：进入攻击范围后发动攻击
- **脱战**：玩家逃离一定距离后回到巡逻

### 3.6 天气/辐射系统

| 天气 | 效果 |
|------|------|
| 晴天 | 无特殊效果 |
| 夜晚 | 视野降低，敌人攻击力 +20% |
| 辐射沙尘 | 每秒 -1 HP（无防护情况下）|
| 酸雨 | 每秒 -2 HP（极少出现）|

### 3.7 剧情系统

- **碎片叙事**：通过物品描述、环境细节讲述背景
- **NPC 对话**：与 NPC 交互获取信息和任务
- **结局分支**：根据存活状态、是否完成支线、Boss 战结果决定结局

## 4. 关卡设计

### 4.1 Ch1：沦陷之城

**时长**：30 分钟
**地图大小**：40x30 格（1格 = 32px）
**资源密度**：高（教学期，给玩家熟悉资源）

**目标**：
1. 熟悉移动和采集
2. 收集 5 木材 + 3 石材
3. 找到避难所入口

**敌人**：丧尸 x5，散布在地图各处

**NPC**：无（教学关单人）

**Boss**：无

---

### 4.2 Ch2：辐射荒原

**时长**：40 分钟
**地图大小**：60x40 格
**资源密度**：中（开始稀缺）

**目标**：
1. 穿越荒原
2. 找到 3 个避难点休息（恢复状态）
3. 找到通往聚落的入口

**敌人**：变异兽 x10，沙尘暴（随机扣血）

**天气机制**：辐射沙尘每 30 秒出现一次，持续 10 秒

**Boss**：无

---

### 4.3 Ch3：避难者聚落

**时长**：30 分钟
**地图大小**：30x30 格
**资源密度**：低（这里资源靠交易）

**目标**：
1. 与 NPC 对话
2. 完成 1 个支线任务（获得钥匙）
3. 用钥匙打开地下通道

**NPC**：3 个可交互 NPC
- 老医生（给任务）
- 商人（交易资源）
- 神秘人（剧情关键人物）

**商店**：
- 食物：5 石材 = 1 食物
- 医疗包：10 石材 = 1 医疗包
- 弹药：5 零件 = 10 弹药

**Boss**：无

---

### 4.4 Ch4：最后的渡口

**时长**：20 分钟
**地图大小**：50x35 格
**资源密度**：低

**目标**：
1. 找到 3 把钥匙碎片
2. 打开通往渡口的大门
3. 击败 Boss
4. 抵达撤离点

**敌人**：末日守卫 x5，精英怪 x2

**Boss：末日守卫（Guardian of Ash）**
- HP: 500
- 攻击模式：重击 + 召唤小怪
- 弱点：躲避后反击窗口

**结局**：
- 击败 Boss + 撤离 → 英雄结局
- 未击败 Boss + 撤离 → 普通结局
- HP ≤ 0 → 死亡结局

- 可交互物品上方：显示 "E" 图标

## 7. 技术规格

### 7.1 引擎配置

- 引擎：Godot 4.3+
- 渲染：2D（像素风，关闭抗锯齿）
- 分辨率：960x540（像素完美）
- 目标帧率：60 FPS
- 导出平台：Mac / Windows / Web

### 7.2 像素规范

- 基础像素：64x64 精灵
- 瓦片：32x32
- 角色：16x16 或 32x32
- UI：8x8 最小单位

### 7.3 存档系统

- 自动存档（每个章节结束）
- 手动存档（任意时刻）
- 存档内容：玩家位置、物品、背包、章节进度

## 8. 验收标准

### 8.1 可玩标准

- [ ] 玩家可以正常移动、采集、战斗
- [ ] 饥饿/体力系统正常运行
- [ ] 可以从 Ch1 通关到 Ch4
- [ ] Boss 可被击败
- [ ] 至少一个结局可触发

### 8.2 质量标准

- [ ] 无恶性 Bug（崩溃、死循环）
- [ ] 帧率稳定在 30 FPS 以上
- [ ] UI 可正常交互
- [ ] 音效/BGM 正常播放

### 8.3 上线标准

- [ ] 完整可玩通关机
- [ ] 有标题画面和游戏结束画面
- [ ] 有基本教程
- [ ] 截图和预告片准备完毕
