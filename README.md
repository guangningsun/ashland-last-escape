# Ashland: Last Escape（灰烬大陆：最后的渡口）

> 像素风末日生存游戏，一个人 + AI 工具链开发

---

## 🎮 游戏概述

**类型：** 2D 俯视角像素风末日生存游戏
**时长：** 中型，完整流程约 2 小时
**平台：** itch.io / Steam
**引擎：** Godot 4.x

**一句话：** 末日降临，你是最后一批幸存者，必须穿过三片废土，找到最后的撤离点。

---

## 📐 游戏结构（4章节）

| 章节 | 时长 | 名称 | 核心内容 |
|------|------|------|---------|
| Ch1 | 30min | 沦陷之城 | 教学 + 城市探索 + 基础资源 |
| Ch2 | 40min | 辐射荒原 | 资源匮乏 + 恶劣环境 + 敌人增多 |
| Ch3 | 30min | 避难者聚落 | NPC 互动 + 支线 + 升级装备 |
| Ch4 | 20min | 最后的渡口 | Boss 战 + 撤离 + 结局 |

---

## 🎮 核心系统

- [ ] **移动** — WASD，俯视角 8 向移动
- [ ] **资源采集** — 木材/石材/食物/医疗/零件
- [ ] **背包** — 有限格，策略取舍
- [ ] **饥饿/体力** — 持续消耗，不管理会死
- [ ] **合成系统** — 收集素材 → 做工具/武器/药品
- [ ] **战斗系统** — 近战 + 远程（弓箭/枪械）
- [ ] **天气/辐射** — 白天安全/夜晚危险
- [ ] **剧情系统** — NPC 对话 + 物品描述 + 碎片叙事
- [ ] **撤离机制** — Ch4 找到撤离点即通关

---

## 🎨 美术风格

- 分辨率：64x64 像素精灵
- 色系：沙黄 + 灰褐 + 暗红（末日废土）
- UI：简约，半透明 HUD

---

## 🛠️ 技术栈

- **引擎：** Godot 4.x（GDScript）
- **AI 代码：** Qwen Coder 30B（昇腾 910B）
- **AI 美术：** Ideogram / Stable Diffusion
- **AI 音乐：** Suno / Udio
- **AI 对话：** vLLM Ascend（NPC 对话生成）
